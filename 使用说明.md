# 使用说明

## 工具功能

这个工具可以把音频文件自动识别成字幕，并按你指定的翻译情景生成一种你选择的字幕文件：

- `original.srt`：原文字幕
- `translation.srt`：中文字幕
- `bilingual.srt`：双语字幕

双语字幕格式固定为：

```text
第一行：中文翻译
第二行：原文
```

## 运行前准备

### 1. 了解自动清理机制

程序每次启动新任务前，会自动清空：

- `input/`
- `output/`

所以：

- 不要把长期保存的音频放进 `input/`
- 不要把重要结果长期留在 `output/`
- 更推荐把待处理音频放在项目外的位置，运行时直接输入文件完整路径

### 2. 准备音频文件

你可以把音频放在项目外任意位置，也可以临时放进 `input/`。

支持常见格式：

- `mp3`
- `wav`
- `m4a`

例如：

```text
D:\Audio\videoplayback (2).m4a
```

### 3. 配置 API Key

项目根目录下有一个 `.env` 文件。

打开它，确认下面这一行已经填写：

```text
LLM_API_KEY=你的DeepSeek_API_Key
```

如果已经写好，就不用再改。

## 使用方法

### 方法一：命令行运行

在项目根目录打开终端，执行：

```bash
python main.py
```

程序会提示你输入三项内容：

```text
音频路径
翻译情景
字幕类型数字
```

示例：

```text
音频路径: D:\Audio\videoplayback (2).m4a
翻译情景: 排球比赛解说
请输入数字 [1/2/3]: 3
```

### 方法二：双击脚本运行

直接双击项目里的 `run.bat`。

双击后会自动：

1. 检查 Python 是否可用
2. 进入项目目录
3. 启动 `main.py`
4. 按提示输入音频路径、翻译情景和字幕类型数字

## 生成结果

处理完成后，字幕文件会输出到 `output` 目录。

程序每次只输出一个文件：

- 选 `1`：生成 `output/original.srt`
- 选 `2`：生成 `output/translation.srt`
- 选 `3`：生成 `output/bilingual.srt`

## 推荐翻译情景示例

你可以根据内容输入不同的翻译情景，例如：

- `排球比赛解说`
- `体育赛事`
- `日常对话`
- `动漫字幕`
- `医学讲解`

## 常见问题

### 1. 提示找不到音频文件

检查输入的路径是否正确，建议优先使用完整路径，例如：

```text
D:\Audio\你的文件名.m4a
```

### 2. 提示没有配置 `LLM_API_KEY`

打开项目根目录的 `.env` 文件，填写你的 DeepSeek Key。

### 3. 识别速度较慢

当前默认使用 CPU 模式，兼容性更好，但速度会比 GPU 慢一些。

### 4. 输出字幕不符合想要的风格

重新运行一次，换一个更明确的翻译情景即可。

例如把：

```text
日常对话
```

改成：

```text
体育赛事解说
```

## 当前默认配置

项目目前已经调整为更稳妥的本地配置：

- Whisper 模型：`tiny`
- 运行设备：`cpu`
- 计算类型：`int8`

这样更适合大多数 Windows 机器直接运行。
